DIMSpec Table Explorer Parameters
=================================

This file lists all available options in the "Choose a table or view" dropdown, categorized by their function.

1. Core Data Tables (핵심 데이터)
---------------------------------
Primary tables storing the main scientific data.
- compounds: Information about chemical compounds (PFAS).
- samples: Information about the physical samples analyzed.
- peaks: Detected peaks in the mass spectrometry data.
- ms_data: Raw mass spectrometry data points (m/z, intensity).
- peak_data: Detailed data associated with specific peaks.
- peak_spectra: Spectral data linked to peaks.
- qc_data: Quality control data.

2. Reference & Metadata Tables (참조 및 메타데이터)
---------------------------------------------------
Tables defining methods, equipment, and standards.
- chromatography_types: Types of chromatography used.
- column_chemistries: Information about column chemistry.
- detectors: Types of detectors used.
- mass_analyzers: Types of mass analyzers.
- mobile_phases: Solvents/mixtures used in the mobile phase.
- ms_methods: Mass spectrometry methods.
- qc_methods: Quality control methods.
- method: General method descriptions.
- contributors: Data contributors and authors.
- element_isotopes: Isotopic information for elements.
- exact_masses: Exact mass reference values.
- opt_ums_params: Optimization parameters.
- version_history: Database version tracking.
- logs: Application or data processing logs.

3. Normalization Tables (정규화 테이블 - prefix: norm_)
-------------------------------------------------------
Tables used for standardizing vocabulary and values across the database.
- norm_chromatography_types
- norm_column_chemistries
- norm_contributors_affiliation
- norm_contributors_name
- norm_detectors
- norm_fragment_sources
- norm_mass_analyzers
- norm_qc_methods_name
- norm_qc_methods_reference
- norm_sample_classes
- norm_source_types
- norm_vendors
- norm_voltage_units

4. Relationship & Alias Tables (관계 및 별칭)
---------------------------------------------
Tables handling many-to-many relationships and alternative names.
- additive_aliases
- annotated_fragments
- carrier_additives
- carrier_aliases
- carrier_mix
- carrier_mix_collection
- carrier_mixes
- compound_aliases
- compound_fragments
- fragment_aliases
- fragment_sources
- sample_aliases
